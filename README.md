# Airline Passenger Satisfaction Predictor

## Overview
Machine learning and deep learning-based prediction system for airline passenger satisfaction using ANN, Random Forest, XGBoost, and Stacking models.

## Features
- Data preprocessing pipeline
- Exploratory Data Analysis (EDA)
- Feature engineering
- ANN model using PyTorch
- Streamlit web application
- Real-time prediction system
- Model comparison and evaluation

## Technologies Used
- Python
- Scikit-learn
- PyTorch
- Pandas
- NumPy
- Streamlit
- XGBoost

## Model Performance
- Accuracy: 96%
- ROC-AUC: 0.995

## Run Project

pip install -r requirements.txt

streamlit run app.py
